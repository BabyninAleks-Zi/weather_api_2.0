# Получите погоду из терминала

![](images/example.png)

Это код для сотрудников международной компании "ООО Терминальная стадия".
При запуске кода в терминале Вы сможете получить данные о погоде в
зависимости от указанного местоположения или определенного города по
всему миру.

***

### Как установить

Прежде чем начать работу с кодом необходимо иметь установленный Python, 
а так же установить библиотеку из файла `requests` 

Для установки библиотеки `requests`, в консоли 
напишите команду и дождитесь установки библиотеки:

```
$ pip install requests
```

Документацию к библиотекам можете изучить на сайте 
[PyPI](https://pypi.org/project/requests/).

***

### Как использовать

Данный код работает просто, необходимо в консоли ввести команду:

```
$ python weather_api.py
```

После ввода указать название города по которому хотите получить 
информацию о погоде на ближайшие 2 дня.

Болле подробно о коде можете посмотреть здесь:

```Python
def generate_weather():
    location = input('Введите город: ') # Переменная input для ввода данных
    url = 'https://wttr.in/{}'.format(location) # URL адрес API сервиса погоды wttr.in
    params = {				# параметры отображения теста, единиц измерения и языка,
        'pnqmMT': '',		# если есть желание сделать по своему, можете их поменять
        'lang': 'ru'		# информацию о параметрах можете посмотреть тут https://wttr.in/:help
    }
    
    try:
        response = requests.get(url, params=params) # Запрос URL
        response.raise_for_status() # Запрос статуса
        print(response.text) # Отображение результата запроса
```
